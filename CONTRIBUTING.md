## Development

1.  Install dependencies: `npm install`
2.  Run dev server: `npm run dev`
3.  Build: `npm run build`

## Code Style

- Use TypeScript for safety.
- Prefer functional components with hooks.
- Use Tailwind CSS for styling via utility classes.
